# ユビキタス言語（Ubiquitous Language）

## 概要

このドキュメントは、注文管理システムにおけるユビキタス言語を定義します。
開発者とドメインエキスパート（業務担当者）が同じ言葉で会話し、コードにもこの用語を使用します。

---

## 用語一覧

| 日本語 | 英語 | 定義 |
|--------|------|------|
| 顧客 | Customer | 商品を購入する人。顧客IDで一意に識別される |
| 注文 | Order | 顧客が購入を希望する商品の集まり。注文IDで一意に識別される |
| 注文明細 | OrderLine | 注文内の1つの商品と数量の組み合わせ |
| カート | Cart | 注文確定前の商品一時保管場所 |
| 商品 | Product | 販売対象の品物。商品IDで一意に識別される |
| 在庫 | Stock | 倉庫に保管されている商品の数量 |
| 支払い | Payment | 注文に対する代金の決済。金額・決済方法・決済状態を持つ |
| 出荷 | Shipment | 注文された商品を顧客へ発送すること |
| 入荷待ち | Backordered | 在庫不足により商品の到着を待っている状態 |

---

## 注文ステータス

| ステータス | 英語 | 説明 |
|------------|------|------|
| 下書き | Draft | 注文作成直後の状態。まだ確定していない |
| 確定済み | Confirmed | 注文が確定し、在庫引当待ちの状態 |
| 支払済み | Paid | 注文の決済が確定した状態。 |
| 出荷済み | Shipped | 商品の出荷が確定した状態。 |
| 配達完了 | Delivered | 商品の配達が確定した状態。 |
| キャンセル | Cancelled | 注文がキャンセルされた状態。 |

---

## ドメインアクション（動詞）

| 日本語 | 英語 | 説明 |
|--------|------|------|
| 注文を確定する | Confirm Order | カートの内容を正式な注文として確定する |
| 在庫を引き当てる | Allocate Stock | 注文に対して在庫を確保する |
| 支払いを処理する | Pay | 注文の支払いを処理する |
| 出荷する | Ship | 商品を顧客に発送する |
| キャンセルする | Cancel | 注文を取り消し、関連する引当を解除する | 

---

## 用語間の関係

```
Customer ──1:N──▶ Order ──1:N──▶ OrderLine ──N:1──▶ Product
                    │
                    ├──1:1──▶ Payment
                    │
                    └──1:1──▶ Shipment

Product ──1:1──▶ Stock
```
